# Testes Automáticos com Cypress – Verificação e Validação (V&V)

Este repositório reúne os casos de testes automatizados desenvolvidos com **Cypress** para a disciplina de **Verificação e Validação de Software (V&V)**.

## 📖 Sobre o projeto

Os testes foram implementados para validar funcionalidades da aplicação **DemoBlaze**, automatizando cenários de teste elaborados durante a disciplina, com base em técnicas de teste de caixa preta.

## 🛠️ Tecnologias utilizadas

- Cypress
- JavaScript
- Node.js

## 📂 Estrutura

Os casos de teste estão organizados em arquivos individuais, identificados pelo código do caso de teste:


## ▶️ Como executar

1. Clone o repositório:

```bash
git clone https://github.com/israellevyzz/Testes-automaticos-com-cypress-V-V.git
```

2. Entre na pasta do projeto:

```bash
cd Testes-automaticos-com-cypress-V-V
```

3. Instale as dependências:

```bash
npm install
```

4. Execute os testes:

Modo interativo:

```bash
npx cypress open
```

Ou pelo terminal:

```bash
npx cypress run
```

## 🎯 Objetivo

- Automatizar casos de teste da aplicação DemoBlaze.
- Validar funcionalidades da interface.
- Aplicar conceitos de Verificação e Validação de Software utilizando o Cypress.
